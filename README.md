# Ecommerce API

## Tecnologias utilizadas

- Node.js
- TypeScript
- Express
- MongoDB

## Rotas

- GET /users - retorna os usu치rios salvos no banco
- GET /me - retorna o usu치rio vinculado com o token
- POST /refresh - refresh token
- POST /register - cria um usu치rio
- DELETE /users/:id - deleta um usu치rio

- GET /products - retorna os produtos salvos no banco
- POST /products - cria um produto
